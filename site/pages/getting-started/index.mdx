# Getting started with Submodule
 
## Add submodule dependency

Add submodule to your project using dependency management of your choice

``` bash
yarn add submodule
```

## Add submodule configuration

Submodule uses `esbuild-register` to process typescript file. You can use either `.js` or `.ts`. Create `submodule.ts` file



## Create routes directory

Submodule, by default, will look for all exported functions in the `routes` directory

<Tabs items={['submodule.ts', 'routes/echo.ts']}>
  <Tab>
    ``` typescript
    import { Submodule } from 'submodule'

    export default <Submodule> {}
    ```
  </Tab>
  <Tab>
    ``` typescript

    ```
  </Tab>
